## 🎯 בדיקה משותפת (כל Roles)

### i18n Switching (עברית ↔ אנגלית)
```
כל משתמש יבדוק:
- [ ] התחברות בעברית
- [ ] לחיצה על EN
- [ ] כל טקסט משתנה לאנגלית
  - Navigation items
  - Button text
  - Table headers
  - Error messages
- [ ] חזרה ל-עברית
- [ ] כל משתנה חזרה
```

### Responsive Design
```
כל משתמש יבדוק בגדלים:
- [ ] 375px (Mobile) — כל עמודים
  - Sidebar collapse/drawer
  - Tables → Cards
  - Buttons גדולים
  - No horizontal scroll
- [ ] 768px (Tablet) — דשבורד + דף אחד
- [ ] 1280px (Desktop) — desktop layout
```

### Performance & Errors
```
כל משתמש יבדוק:
- [ ] בדיקת console (F12) — אין errors
- [ ] Loading states — spinners תצוגה
- [ ] Error messages — תצוגה כראוי
- [ ] Network requests — no 401/403 (afer auth fix)
```
