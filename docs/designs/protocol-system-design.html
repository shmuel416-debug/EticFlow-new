     │
     ▼
[Create Protocol] ─── auto-fills from meeting agenda
     │
     ▼
[ProtocolDetailPage] ── DRAFT status
     │
     ├─ Edit sections (heading + content textareas)
     ├─ Save draft
     └─ Finalize → PENDING_SIGNATURES
                          │
                          ▼
              [Request Signatures]
              ── select signers (meeting attendees)
              ── sends email with unique token per signer
                          │
                          ▼
              [ProtocolSignPage] ─── PUBLIC, no auth
              ── signer opens email link
              ── sees: protocol title, date, their name
              ── action: Sign ✓ or Decline ✕
              ── token invalidated after use
                          │
                    ┌─────┴──────┐
                    ▼            ▼
                 SIGNED       DECLINED
                    │
                    ▼ (all signed)
              Protocol → SIGNED status
                    │
                    ▼
              [Download PDF] ── available any time
